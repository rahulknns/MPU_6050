◆ disableSleepMode()

void MPU6050::disableSleepMode ( )

Disable the sleep mode.

Note
The MPU6050 device will enter sleep mode when the SLEEP bit is set to 1 in the PWR_MGMT_1 register. In sleep mode, only the serial interface and internal registers remain active, allowing for a very low standby current. Clearing the SLEEP bit in PWR_MGMT_1 wakes the device.
